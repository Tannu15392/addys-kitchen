# 🍽️ Addy's Kitchen – Restaurant Website

A modern restaurant landing page built using **HTML, CSS, and JavaScript**.
This project demonstrates a clean restaurant UI with responsive layout, food sections, gallery, and contact information.

---

## 🌐 Live Demo

https://tannu15392.github.io/addys-kitchen/

---

## 📌 Features

* Responsive design (mobile friendly)
* Elegant restaurant landing page
* Food menu showcase
* Image gallery
* Contact & location section
* Smooth UI layout

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript

---

## 📂 Project Structure

```
addys-kitchen/
│
├── index.html
├── style.css
├── script.js
└── images/
```

---

## 🎯 Purpose

This project was created as a **frontend development practice project** and a **demo website for a local restaurant (Addy's Kitchen, Sitarganj)** to showcase how small businesses can build a professional online presence.

---

## 🚀 How to Run Locally

1. Clone the repository

```
git clone https://github.com/tannu15392/addys-kitchen.git
```

2. Open the folder

3. Run `index.html` in your browser
